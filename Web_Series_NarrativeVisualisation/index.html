Trends in Web Series over the Decades - A Narrative Visualization

For this Narrative Visualization Project, I have analyzed a web series dataset to study which genres and ratings have appealed to viewers' interests over the years. The original dataset for this Narrative Visualization has been taken from: https://www.kaggle.com/datasets/muralidharbhusal/50000-imdb-tv-and-web-series. The dataset was cleaned to remove irrelevant data and ensure data integrity. We also added some additional fields to the dataset to make it more informative. The dataset includes fields such as series title, runtime, genre, rating, decade, release year, end year, duration, cast, and synopsis. The flow of this Narrative Visualization is as follows:

  • Scene 1: Web Series Popularity over the Decades
  • Scene 2: Web Series Distribution across top-10 Genres for each decade
  • Scene 3: Average Rating of Web Series for top-10 Genres for each decade
  • Scene 4: Evolution of User Preferences over the Decades based on our Analysis

Bar Chart of Web Series Popularity over the Decades

Insights

This interactive visualization displays a bar chart representing the number of unique web series released in each decade. The data is aggregated to show the count of distinct series titles for each decade, providing an overview of the trends in web series production over time.

The visualization reveals the following insights:

  • There is a noticeable upward trend in the number of web series released over the decades. This suggests a growing interest in web series as a form of entertainment, likely due to the rise of streaming platforms and digital media consumption.
  • The first two decades (1930-1940, 1940-1950) show significantly fewer web series than the other decades. This is expected since web series as a format gained popularity more recently, and the availability of data from earlier decades might be limited.
  • The data for the last decade (2020-2030) is incomplete, with values only till 2023. This partial data might not accurately reflect the total production for this period, especially since the decade is ongoing.

Conclusion

The interactive bar chart effectively highlights the evolution and increasing popularity of web series over time. Since there is limited data for before 1940 and limited data after 2020, we shall not include them in our analysis. Moving forward, we shall analyze the trends in web series genres and ratings for the decades in the range of 1950 to 2020 to gain further insights into the web series landscape.
