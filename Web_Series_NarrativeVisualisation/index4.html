Trends in Web Series over the Decades - A Narrative Visualization

For this Narrative Visualization Project, I have analyzed a web series dataset to study which genres and ratings have appealed to viewers' interests over the years. The original dataset for this Narrative Visualization has been taken from: https://www.kaggle.com/datasets/muralidharbhusal/50000-imdb-tv-and-web-series. The dataset was cleaned to remove irrelevant data and ensure data integrity. We also added some additional fields to the dataset to make it more informative. The dataset includes fields such as series title, runtime, genre, rating, decade, release year, end year, duration, cast, and synopsis. The flow of this Narrative Visualization is as follows:

  • Scene 1: Web Series Popularity over the Decades
  • Scene 2: Web Series Distribution across top-10 Genres for each decade
  • Scene 3: Average Rating of Web Series for top-10 Genres for each decade
  • Scene 4: Evolution of User Preferences over the Decades based on our Analysis

Line Chart Evolution of User Preferences Over the Decades

Insights

This interactive visualization displays a line chart where each line corresponds to a particular Genre and plots the number of unique web series released in each decade.

The visualization reveals the following insights:

  • "Drama" Genre seems to have gained very high popularity in the recent years, surpassing "Comedy".
  • "Drama" Genre leads with highest count of 1283 releases in the decade 2010-2020.
  • "Comedy" Genre was the leading genre till 2000.
  • Genres such as "Western", "Family" have lost popularity over time.
  • Genres such as "Action", "Animation" and "Crime" are slowly gaining popularity.

Conclusion

This Narrative Visualization provides us a comprehensive understanding about the kind of Web Series Released over the years. It allows us to observe shifts in genre popularity over time, highlighting genres that have gained preference and those that have declined in popularity. By analyzing these trends, we can gain insights into changing viewer interests and the evolving landscape of web series across different decades.
