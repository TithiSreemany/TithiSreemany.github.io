Trends in Web Series over the Decades - A Narrative Visualization

For this Narrative Visualisation Project, I have analyzed a web series dataset to study which genres and ratings have appealed to viewers' interests over the years. The original dataset for this Narrative Visualisation has been taken from: https://www.kaggle.com/datasets/muralidharbhusal/50000-imdb-tv-and-web-series. The datset was cleaned to remove irrelevant data and ensure data integrity. We also added some additional fields to the dataset to make it more informative. The dataset includes fields such as series title, runtime, genre, rating, decade, release year, end year, duration, cast, and synopsis. The flow of this Narrative Visualisation is as follows:

  • Scene 1: Web Series Popularity over the Decades
  • Scene 2: Web Series Distribution across top-10 Genres for each decade
  • Scene 3: Avergae Rating of Web Series for top-10 Genres for each decate
  • Scene 4: Evolution of User Preferences over the Decades based on our Analysis

Insights

In the third scene of the Interactive Visualisation, we display a bar chart showing the average rating of web series across the top-10 Genres for a particular decade.

  • The chart is interactive and allows the user to select the decade of interest ranging from 1950-1960 to 2010-2020 and the bar chart is updated dynamically according to the chosen year.
  • The x-axis shows the Top-10 Genres sorted in descending order according to the number of web series released.
  • The color-scheme to represent the Web Series Genres remain consistent across the decades for easier comparison.
  • The chart is also annotated with the average rating of web series in each genre and their average rating.
  • We observe that across the decades, the average rating for the Top-10 Genres fall in the range of 6.5 to 8.5 which no specific trends.

The chart is useful for verifying whether the Web Series being made are appealing to the users. We observe that across the decades, the average rating for the Top-10 Genres fall in the range of 6.5 to 8.5 which no specific trends across the Genres. Hence we can conclude that the quality of the Web Series has been consistent over the years.
